## 📚 Sobre

Este repositório é um **laboratório de estudos em Python**, focado no aprendizado prático de:

* 📦 Uso de bibliotecas
* 🧠 Programação Orientada a Objetos (POO)

Os códigos aqui são experimentos e exercícios que ajudam a evoluir na linguagem e na organização de projetos.

---

## 🚀 O que você vai encontrar

* Scripts de estudo em Python
* Exemplos utilizando bibliotecas
* Implementações de POO
* Testes e práticas de conceitos

---

## 🎯 Objetivo

Consolidar o conhecimento em Python e evoluir para projetos mais estruturados utilizando **POO e boas práticas de desenvolvimento**.
